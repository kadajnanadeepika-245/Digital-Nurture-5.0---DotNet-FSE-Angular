## 📖 About

This repository showcases my learning journey through the **Cognizant Digital Nurture 5.0 Deep Skilling Program (DotNet FSE - Angular Track)**. It contains hands-on exercises, coding assignments, and projects covering Full Stack Development concepts using Microsoft technologies and Angular.

The goal of this repository is to strengthen software engineering fundamentals, backend development, frontend development, database management, testing, and modern development practices.

## 🎯 Learning Outcomes

- Understand SOLID Principles and Design Patterns
- Master Data Structures and Algorithms
- Develop and optimize SQL Server databases
- Write Unit Tests using NUnit and Moq
- Build applications using Entity Framework Core
- Create RESTful APIs using ASP.NET Core
- Understand Microservices Architecture
- Develop responsive web applications using Angular
- Work with Git, GitHub, Docker, and CI/CD concepts
